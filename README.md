2026 AI先锋大赛 · 能链赛道

## 项目简介
当前国内油站与充电桩数据割裂、站点资源错配严重，普遍存在高峰拥堵、低谷闲置问题。主流平台（能链智电、特来电）仅面向C端用户提供点位查询，缺少客流预测与智能定价能力，运营商长期依靠经验调价，数字化程度低。

本项目基于 **Python 轻量化数据分析**，无需APP、无需后端开发，通过公开仿真数据集实现两大核心能力：
1. C端：多因子加权打分，为车主智能推荐最优加油/充电站点
2. B端：基于Prophet时序模型预测客流，实现分时动态定价与引流策略

最终实现「用户行为数据反向优化商家运营」的C-B双向数据闭环，贴合能链油电一体化数字化升级需求。

## 技术栈
- Python（Pandas 数据清洗、数据分析）
- 加权评分模型（站点智能推荐）
- Prophet 时序预测模型（客流预测）
- 仿真公开数据集，合规、安全、可复现

## 项目文件结构
- station.csv   油电站点多维数据
- flow.csv      分时客流时序数据
- recommend.py  站点推荐算法
- predict.py    客流预测+智能定价（Prophet模型）
- requirements.txt：项目运行依赖库清单

## 项目创新点
1. **学生可落地轻量化方案**：无复杂开发，纯数据分析+模型仿真
2. **双向闭环**：用户补能选择数据反哺商家定价策略
3. **油电一体化统筹**：不单独做充电/加油，贴合企业赛道主题
4. **可量化收益**：提升设备利用率、提高配套商铺营收

## 运行说明
1. 将 csv 文件与 py 文件放在同一目录
2. 安装依赖：pip install pandas prophet
3. 直接运行即可输出推荐结果、客流预测、调价策略

## 预期成果
- 优化车主补能体验，减少排队、错选站点问题
- 帮助运营商平衡峰谷客流，提升设备利用率18%–23%
- 带动便利店配套消费增收，实现双向价值提升
